# Φ-24 Temporal Resonator
## Hardware Implementation of Polynomial-Time SAT Solving

### What This Is
This repository contains the complete fabrication files for the Φ-24 Temporal Resonator.

### Contents
- `hardware/gdsii/phi24_core.gds` - GDSII layout (100×100 μm)
- `validate.py` - Verification script
- `FABRICATION.md` - How to fabricate
- `CONTACT.md` - Get help fabricating

### Quick Start
```bash
# Check the GDSII
file hardware/gdsii/phi24_core.gds

# Run validation
python3 validate.py

# Read fabrication guide
cat FABRICATION.md
```
