# How to Fabricate the Φ-24

## 1. Requirements
- Foundry access (Intel, TSMC, SkyWater, etc.)
- MBE for Bi₂Se₃/NbSe₂ growth
- Cryogenic testing (20 mK)

## 2. Files Provided
- `phi24_core.gds` - Complete layout
- 100×100 μm chip
- 21-layer Fibonacci superlattice
- 33×33 Josephson junction array

## 3. Steps
1. Submit GDSII to foundry
2. Use MBE with golden ratio thickness
3. Package at 20 mK
4. Test at 1.485 MHz
